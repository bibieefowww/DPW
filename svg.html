SVG (Scalable Vector Graphics)

Kesimpulan: SVG digunakan untuk membuat gambar berbasis vektor di halaman web seperti lingkaran, persegi, dan poligon. Keunggulan SVG adalah kualitas gambar tidak pecah saat diperbesar dan dapat dimanipulasi menggunakan atribut atau CSS.
