Kesimpulan: Beberapa tipe input seperti datetime-local, month, dan email memiliki dukungan berbeda di setiap browser. Sebagian browser menampilkan UI khusus, sementara lainnya hanya berupa input biasa dengan validasi.

Uji coba 6






Kesimpulan: Input type image berfungsi sebagai tombol submit berbentuk gambar. Input type file digunakan untuk mengunggah file dari komputer. Input type hidden menyimpan data yang tidak ditampilkan kepada pengguna tetapi tetap dikirim saat form disubmit.

Uji coba 7






Kesimpulan: Input type number membatasi angka sesuai nilai min dan max. Input type range menampilkan slider untuk memilih nilai dalam rentang tertentu. Jika nilai min diubah (misalnya -50), maka pengguna bisa memilih nilai yang lebih kecil sesuai batas baru.

Uji coba 8









Kesimpulan: Input type url digunakan untuk memasukkan alamat website dengan validasi format URL. Input type search digunakan untuk pencarian. Input type tel digunakan untuk nomor telepon dan dapat diberi pola (pattern) agar format sesuai.

Uji coba 9






Kesimpulan: Input type time digunakan untuk memilih waktu (jam dan menit). Input type week digunakan untuk memilih minggu dalam satu tahun. Dukungan tampilan bisa berbeda di setiap browser, ada yang menampilkan picker khusus, ada juga yang hanya input biasa.
