Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama action_page.php.

Kesimpulan: Tombol reset akan mengembalikan input ke nilai awal. Tombol kirim akan mengirim data ke server sesuai atribut action. Atribut method menentukan cara pengiriman data (GET atau POST), dan target menentukan di mana hasil ditampilkan (_self, _blank, dll).
